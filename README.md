# Retirement Monte Carlo (Simple)

This version is intentionally **single-file** and **no-build**.

## Use on GitHub Pages
1. Upload `index.html` to your repo root
2. Repo Settings → Pages → Deploy from branch → `main` → `/ (root)`
3. Your site will run at: `https://<user>.github.io/<repo>/`

## Notes
- Uses Chart.js via CDN.
- Saves scenarios in LocalStorage in your browser.
